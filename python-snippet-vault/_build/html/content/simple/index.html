🐍 Módulos Built-in do Python#

O Python oferece uma série de módulos integrados que facilitam a vida dos desenvolvedores ao fornecer funcionalidades essenciais para manipulação de arquivos, estrutura de dados, operações matemáticas, manipulação de tempo e muito mais.

Este guia contém exemplos práticos e explicações sobre os seguintes módulos:

📂 Lista de Módulos#

🔹 Manipulação de Arquivos e Diretórios#

  • 📁 OS – Interação com o sistema operacional

  • 📁 OS Path – Operações com caminhos de arquivos e diretórios

  • 📁 Shutil – Manipulação avançada de arquivos e diretórios

  • 📁 CSV – Leitura e escrita de arquivos CSV

  • 📁 JSON – Manipulação de arquivos JSON

🔹 Manipulação de Dados#

  • 📊 Collections – Estruturas de dados avançadas (deque, Counter, namedtuple, etc.)

  • 📊 Re (Regex) – Expressões regulares para busca e manipulação de strings

  • 📊 Logging – Registro de logs para depuração e monitoramento

🔹 Operações Matemáticas e Estatísticas#

  • 🧮 Math – Funções matemáticas básicas (logaritmos, seno, cosseno, etc.)

  • 🧮 Time – Manipulação de datas e tempos

  • 🧮 Datetime – Funções avançadas para trabalhar com datas

🔹 Fluxo de Controle e Execução#

  • ⚙️ Sys – Acesso a parâmetros e funções do interpretador Python

  • ⚙️ Argparse – Manipulação de argumentos de linha de comando

  • ⚙️ Functools – Funções de programação funcional e otimizações

🔹 Geração de Dados Aleatórios#

  • 🎲 Random – Geração de números aleatórios e embaralhamento de listas
